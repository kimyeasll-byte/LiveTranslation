# Live Translation 사용자 매뉴얼

본 매뉴얼은 Live Translation (실시간 동시통역 앱)의 사용 방법을 안내합니다.

## 1. 시작하기 (초기 설정)

앱에 접속하면 가장 먼저 **로비(Lobby)** 화면이 나타납니다.

1. **이름 입력**: 대화방에서 사용될 나의 이름을 입력합니다.
2. **언어 설정**: 
   - **원본 언어 (나의 언어)**: 내가 주로 말할 언어를 선택합니다.
   - **번역 언어 (상대방 언어)**: 상대방이 말할 언어이자, 내가 듣고 싶은 번역 언어를 선택합니다.
3. **엔진 설정 (선택 사항)**:
   - **음성 인식 (STT)**: 기본적으로 매우 정확한 `Groq (Whisper)` 엔진이 설정되어 있습니다.
   - **번역 엔진**: 무료로 사용 가능한 `MyMemory`가 기본 설정되어 있습니다. 더 높은 품질을 원하신다면 `Google Cloud`나 `DeepL`을 선택하고 본인의 API 키를 입력하세요.

## 2. 모드 선택

설정이 완료되면 하단의 두 버튼 중 하나를 선택하여 시작합니다.

### 👥 상대방과 함께 통역 (P2P 모드)
멀리 떨어진 상대방과 각자의 기기로 접속하여 실시간으로 통역 대화를 나눕니다.
- 버튼을 누르면 즉시 번역 대기화면으로 이동합니다.
- 상단에 나타나는 **링크 복사** 또는 **코드 복사** 버튼을 눌러 상대방에게 전달합니다.
- 상대방이 해당 링크를 클릭하거나 코드를 입력하여 접속하면 연결이 완료됩니다.

### 👤 혼자 사용하기 (솔로 모드)
상대방 없이 화면을 띄워두고, 하나의 기기에서 양방향 통역기로 사용합니다. 해외 여행 중 오프라인 대화를 나눌 때 유용합니다.

## 3. 실시간 통역 사용법 (하이브리드 연속 녹음)

방에 입장하면 메신저 형태의 대화창이 나타납니다.

1. 하단의 **마이크(🎙️) 버튼**을 **단 한 번** 클릭합니다.
2. 버튼이 **정지(⏹️)** 모양으로 바뀌고, '자동 녹음 중' 메시지가 나타납니다.
3. **자연스럽게 말씀하세요.** 말하는 도중에는 화면에 회색 글씨로 실시간 타이핑 효과가 나타납니다.
4. **말씀을 끝내고 약 1.5초간 잠시 쉬면**, 시스템이 자동으로 문장을 끊어서 번역을 시작합니다.
5. 번역된 텍스트가 대화창(말풍선)에 나타나고 상대방에게 전송됩니다.
6. 대화가 완전히 끝날 때까지 마이크를 끄실 필요가 없습니다! (종료 시에만 정지 버튼 클릭)

> [!TIP]
> 텍스트로 대화하고 싶다면 우측 하단의 키보드 버튼(⌨️)을 눌러 직접 텍스트를 입력할 수도 있습니다.

## 4. 대화 기록 내보내기 및 AI 요약

중요한 비즈니스 미팅이나 기록이 필요한 대화를 마쳤다면 문서를 다운로드할 수 있습니다.

1. 우측 상단의 **내보내기(⭳)** 아이콘을 클릭합니다.
2. **PDF로 저장** 또는 **Word 문서로 저장**을 선택합니다.
3. "AI 요약을 생성 중입니다..." 라는 메시지가 나타나고 잠시 기다리면 다운로드가 시작됩니다.
4. 다운로드된 문서의 **최상단**에서 전체 회의를 3~4줄로 요약한 **AI 회의 요약**과 향후 진행해야 할 **Action Items**를 확인할 수 있습니다.
